# Eco-Dev: Red de Microestaciones Inteligentes

Bienvenido al repositorio oficial del proyecto Eco-Dev, una plataforma avanzada de monitoreo ambiental desarrollada para la Feria de Ciencias 2026.

## Sobre el Proyecto
Eco-Dev es una solución integral que combina hardware IoT y software web para monitorear la calidad del aire en tiempo real. Nuestro sistema está diseñado para proteger la salud respiratoria detectando gases nocivos y partículas en suspensión en diversos entornos.

### Objetivos
*   Monitoreo en Tiempo Real: Visualización instantánea de telemetría ambiental.
*   Alertas Tempranas: Identificación de picos de contaminación (CO2, Humo, PM2.5).
*   Accesibilidad: Interfaz responsiva y amigable para el usuario final.
*   Análisis de Datos: Capacidad de exportar historiales para estudios detallados.

## Stack Tecnológico
*   Frontend: Angular 19 (Signals, Standalone Components).
*   Visualización: Chart.js para gráficos dinámicos.
*   Backend: Firebase (Realtime Database, Firestore, Auth).
*   Estilos: Vanilla CSS con enfoque "Mobile-First" y modo oscuro/claro.
