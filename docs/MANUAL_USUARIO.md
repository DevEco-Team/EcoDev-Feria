# Manual de Usuario - Dashboard Eco-Dev

## Acceso al Sistema
Para acceder al panel de monitorización, el usuario debe autenticarse mediante su correo electrónico o a través de su cuenta de Google en la página de **Login**.

## Panel General
Este panel es la vista principal de monitoreo en tiempo real:
- **Selector de Estaciones**: Ubicado en la parte superior, permite cambiar entre las distintas microestaciones instaladas (ej: Plaza San Martín, Av. Colón, etc.).
- **Tarjetas de Métricas**: Muestra los valores actuales de CO2, Partículas PM2.5, Humo, Temperatura y Humedad.
- **Gráficos en Vivo**: Visualización histórica de los últimos 50 registros para detectar tendencias y picos de contaminación.

## Historial de Mediciones
Desde el menú lateral, se puede acceder al historial completo:
- **Tabla de Registros**: Permite ver de forma ordenada y secuencial (del reporte más antiguo al más reciente, con números correlativos "Lectura 1", "Lectura 2", etc.) la fecha, hora y datos de telemetría de cada captura. Muestra nombres amigables de las estaciones de monitoreo para mayor claridad.
- **Filtros Avanzados**:
  - **Buscador de Texto**: Permite filtrar en tiempo real escribiendo el ID de la estación, el nombre amigable de la misma, o el estado de calidad de aire.
  - **Selector de Estación**: Filtra la tabla para mostrar únicamente los registros de una estación específica elegida del listado.
  - **Selector de Estado**: Filtra la tabla para mostrar las mediciones que coincidan con una clasificación particular (ej: "Excelente", "Malo", "Moderado", etc.).
- **Exportación**: El botón "Exportar CSV/Excel" descarga un reporte estructurado con todos los datos filtrados en pantalla para su análisis en herramientas de hojas de cálculo como Microsoft Excel.

## Mapa de Calor
Muestra la intensidad de contaminantes distribuida geográficamente, permitiendo identificar "puntos calientes" de mala calidad del aire en la zona de estudio.

## Ajustes y Perfil
El usuario puede modificar sus datos personales, cambiar la contraseña o ajustar las preferencias de notificaciones y tema (Modo Noche/Día) desde la sección de **Ajustes**.
